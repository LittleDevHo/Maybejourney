# Maybejourney

Midjourney UI: An elegantly designed, highly customizable interface, purpose-built to enhance the prompt engineering with ChatGPT.

![](assets/preview.gif)

## Features

- Midjourney with highly customizalbe UI
- User gallery
- ChatGPT prompt helper
